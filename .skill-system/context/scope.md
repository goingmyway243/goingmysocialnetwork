# Project Scope

## In Scope
- Multi-service .NET backend (User, Post, Notification, Message services)
- Angular SPA frontend with modules for each feature area
- Glassmorphism design system with reusable components
- PostgreSQL data modeling and migrations
- Docker containerization for each service
- Unit and integration tests
- API documentation (OpenAPI/Swagger)
- Basic CI/CD pipeline configuration

## Out of Scope (Phase 2+)
- Mobile app (iOS/Android native)
- Machine learning-powered recommendations
- Advanced analytics dashboard
- Blockchain integration
- Multi-language i18n (Phase 2)

## Technical Boundaries
- Single tenant application (multi-tenant in Phase 2)
- No external payment processing (Phase 2)
- No video streaming (Phase 2)
- Max users: 100K in Phase 1

## Dependencies
- .NET 10.0 or higher
- Node.js 22+ for Angular development
- PostgreSQL 13+
- Docker & Docker Compose
