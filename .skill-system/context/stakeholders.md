# Stakeholders

## Development Team
- **Backend Engineers** - Develop microservices, APIs, business logic
- **Frontend Engineers** - Build Angular components, implement UI designs
- **UI/UX Designer** - Define glassmorphism design system
- **DevOps Engineer** - Infrastructure, CI/CD, deployments

## Other Roles
- **Product Manager** - Feature prioritization, roadmap
- **QA Engineer** - Testing strategy, quality assurance
- **Tech Lead** - Architecture decisions, code standards

## Responsibilities by Role
- **Backend**: API contracts, data models, service logic, scalability
- **Frontend**: Component architecture, state management, accessibility
- **Design**: Design tokens, component specs, brand consistency
- **DevOps**: Containerization, deployment automation, monitoring
